工具¶

本章中的函数执行各种实用工具任务，包括帮助 C 代码提升跨平台可移植性，在 C 中使用 Python 模块，以及解析函数参数并根据 C 中的值构建 Python 中的值等等。

  • 操作系统实用工具
  • 系统功能
  • 过程控制
  • 导入模块
  • 数据 marshal 操作支持
  • 解析参数并构建值变量
    • 解析参数
      • 字符串和缓存区
      • 数字
      • 其他对象
      • API 函数
    • 创建变量
  • 字符串转换与格式化
  • PyHash API
  • 反射
  • 编解码器注册与支持功能
    • Codec 查找API
    • 用于Unicode编码错误处理程序的注册表API
  • PyTime C API
    • 类型
    • 时钟函数
    • 原始时钟函数
    • 转换函数
  • 对 Perf Maps 的支持
